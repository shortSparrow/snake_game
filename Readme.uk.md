# Terminal Snake Game (C++)

Консольна реалізація класичної гри "Змійка", написана на C++ з використанням багатопотоковості та ANSI-кодів для оптимізації рендерингу.

## Технічний опис

Програма реалізує ігрову логіку через пряме керування курсором термінала. Замість повного перемальовування екрана (що викликає мерехтіння), оновлюються лише змінені координати (голова, хвіст та їжа).

### Ключові особливості:
* **Concurrency:** Використання `std::thread` для відокремлення ігрового циклу від обробки вводу.
* **Synchronization:** Захист стану гри за допомогою `std::mutex` для потенційного уникнення race condition.
* **Rendering:** Оптимізований вивід через ANSI escape-послідовності.
* **Input Handling:** Неблокуюче читання клавіш (Raw mode) для миттєвої реакції.

---

## Системні вимоги

* **ОС:** Windows (використовуються `windows.h` та `conio.h`).
* **Термінал:** Windows Terminal, PowerShell або CMD з підтримкою VT100 (ANSI).
* **Компілятор:** Підтримка стандарту C++17 або вище.

---

## Керування

Введення команд виконується в реальному часі без підтвердження клавішею Enter.

| Клавіша | Дія |
| :--- | :--- |
| **W** | Рух вгору |
| **S** | Рух вниз |
| **A** | Рух вліво |
| **D** | Рух вправо |
| **ESC** | Вихід з гри |

### 1. Підготовка компонентів
Переконайтеся, що в одній директорії знаходяться наступні файли:
* `main.cpp` — основний код програми.
* `game_settings.h` — файл із налаштуваннями (розміри поля, символи, кольори).
